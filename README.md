
# 🌱 GramaDharini (ग्रामधारिणी)

> **Empowering Rural Students Across India**

GramaDharini is a free, all-in-one platform built to bridge the gap between rural students and career opportunities. It brings together internships, a resume builder, free courses, and hackathon listings — all in one place, completely free.

Built for **Summer Of Codesfest 2.0** by **Team GramaDharini**.

---

## 🎯 Problem Statement

Many talented students in villages and small towns across India lack access to:
- Internship opportunities
- Resume-building tools
- Quality free courses
- Information about hackathons and competitions

**GramaDharini** solves this by bringing all these resources onto a single, free, easy-to-use platform — accessible to students from any state, village, or background.

---

## ✨ Features

| Feature | Description |
|---|---|
| 💼 **Internships** | Browse curated internship listings from companies across India, including remote-friendly roles for rural students. |
| 📄 **Resume Builder** | Fill in your details, preview live, and download a professional resume as a PDF — free, no login required. |
| 📚 **Free Courses** | Access top free courses from YouTube, NPTEL, Google, and Coursera across Web Dev, Data Science, Agriculture, English, and Business. |
| 🏆 **Hackathons** | Stay updated on hackathons, quizzes, design challenges, and business competitions with prize details and registration links. |
| 🔐 **Login / Register** | Simple authentication flow for students to create a free account. |

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- Python 3.x (for the backend/resume PDF feature)

### Running Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/HackIndiaXYZ/summer-of-codesfest-20-team-gramadharini.git
   cd summer-of-codesfest-20-team-gramadharini
   ```

2. **Start the backend** (required for resume PDF downloads)
   ```bash
   cd backend
   python app.py
   ```
   This runs the backend locally at `http://127.0.0.1:5000`.

3. **Open the frontend**
   Open `index.html` in your browser, or serve the root folder with a local server (e.g., VS Code Live Server) for the best experience.
